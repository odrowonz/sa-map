# Заметки по хостингу

Файл **`it-senior.kz.md`** может содержать **пароли, ключи доступа и персональные данные**.

- Не публикуйте его в открытых репозиториях без редактирования.
- Добавьте `it-senior.kz.md` в `.gitignore`, если репозиторий публичный.
- Храните актуальные учётные данные в менеджере паролей.

**БД приложения «Карта системного аналитика»:** параметры подключения — в **`sa-map-database.md`** (содержит пароль; файл **в `.gitignore`**, в репозиторий не попадает). Шаблон без секретов: [`sa-map-database.example.md`](sa-map-database.example.md).

Общая инструкция по развёртыванию: [../HOSTING_INSTALL.md](../HOSTING_INSTALL.md).
