Повтор рендера Markdown (Nunjucks)

Правьте JSON экспорта MD (например {slug}.json — один файл с манифестом и полями Nunjucks) и *.njk в редакторе, затем нажмите прогон. Нужен интернет при первой загрузке (Nunjucks с CDN). Открывайте HTML из папки с распакованным ZIP — пути icons/, files/, pics/ в MD останутся относительными.

Режим Chromium / Edge (удобнее)

Один раз привяжите файлы. Каждый прогон заново читает с диска — после сохранения в редакторе снова нажмите «Прогон Nunjucks», переоткрывать файлы не нужно.
