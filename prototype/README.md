# Статические прототипы

- **`index.html`** — последняя версия прототипа SA Hub (localStorage, частично L1–L2).
- **`index_stage1.html`** — промежуточная версия.

Откройте файл в браузере двойным щелчком или через локальный HTTP-сервер. Для продакшена используйте собранное приложение из каталога `build/` (после появления дистрибутива).
